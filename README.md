# Monitor de Actividad de Protocolos de Red
Este proyecto es un sistema de monitoreo de equipos en red que captura información del sistema como OS, hostname, IP, MAC, fabricante y puertos usados, verifica que MAC e IP no estén duplicadas en la red, almacena los datos en una base MySQL organizada por equipo, fabricante y protocolos, utiliza un cliente que corre como servicio de Windows con ícono en el System Tray, muestra los datos en una interfaz web con clasificación de seguridad por colores y genera reportes y gráficas de protocolos seguros e inseguros, en esencia recoge datos de los equipos, los valida, los centraliza y los muestra de forma visual y segura.
